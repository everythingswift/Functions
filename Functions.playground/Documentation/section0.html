Functions

A function is a chunk of code that can perform a task. A function can return a value or another function or none at all. A function can take a set of input parameters or none. The function acts on the parameters, executes the task and passes on the control to the line next to where it got called from.

Before getting into the details of return types and input parameters, let's understand the basics of function calls in a program. Below this paragraph in blue color is a simple representation of code flow, when a call is made to a function. The program executes through line 5 to 7, then a call is made to the function on line 8. Now the function gets executed and returns the control to line 9 and continues execution.

Defining and Calling Functions:

A function's definition provides the following details.

  • Function name - that is used to call the function
  • Parameters - input values that the function accepts
  • Return Type - the output that the function returns

func functionName (var1:Type, var2:Type2...) -> ReturnType
{
return (ReturnType value)
}

Now when you have to call the above function, you make the call as shown below. The returnedValue captures the value returned by the function 'functionName'.

var returnedValue = functionName(Input Parameters)
