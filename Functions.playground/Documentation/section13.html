In-Out Parameters:

The variable parameters in Swift let you change the parameter's value within the scope of the function. But if you want the change to prevail beyond the scope of the function, then make use of In-Out parameters.

When calling the function using inout parameter, pass the variable name prefixed with '&' to signify that it's an inout parameter. Also make sure the variable being passed is not a constant. There should be no let or var keywords within the definition of function and 'inout' is sufficient to define that the parameter is an inout parameter.
